# Expense Tracker in C++

A console-based Expense Tracker application developed in C++ using Object-Oriented Programming principles. The application helps users manage their expenses, track their budget, and generate monthly expense reports using persistent file storage.

## Features

* Add new expenses
* View all expenses
* Search expenses by ID
* Delete expenses
* Set and track income
* View budget status
* Generate monthly expense reports
* Persistent data storage using text files
* Input validation and error handling

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Vector
* File Handling (`ifstream`, `ofstream`)
